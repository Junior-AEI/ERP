# Requirements

```
node >= 14.0.0
```

# Installation

```bash
./setup.sh
```

# Running

## For the server
```bash
cd server
npm run dev
```

## For the client
```bash
cd client
npm run dev
```

# Launch server tests

To launch server tests, you just have to go to launch the following command :

```bash
cd server
npm run test
```
